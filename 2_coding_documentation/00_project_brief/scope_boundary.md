# 范围边界（Scope Boundary）

## 本次必做（MVP）

- 基础输入：偏好/饮食限制/预算/时间/厨具水平等
- 计划输出：菜谱建议 + 学习路径（分步）+ 采购清单 + 每日/每餐安排
- 数据存储：`localStorage`（最小可用）
- AI：mock 链路跑通（UI → service → mock generator → 展示）

## 本次不做（明确不实现）

- 真实云端模型接入（仅保留接口占位与说明）
- 复杂账号系统/多人协作/云端存储
- 完整知识库构建与在线向量检索（用本地 mock 证明“RAG-ready 架构”即可）

## 风险与应对

- 依赖安装/网络受限：优先保证代码可读与结构清晰，必要时提供离线可运行方案
- 演示时间有限：功能以“可讲清楚链路”为优先，不追求大而全
