# Step 2.0-04 Prompt Log（随机弹窗 + 成功提示音）

## 输入 prompt 原文

请实现首页“随机一道菜”完整弹窗交互，并增加成功提示音。

要求：
1）点击首页右侧“随机一道菜”模块中的按钮后，弹出中心模态层  
2）弹窗包含：
   - 标题“随机一道菜”
   - 说明文案
   - 一张完整菜谱卡片
   - 关闭按钮
   - “再摇一次”
   - “查看详情”
3）点击“再摇一次”刷新弹窗中的菜谱内容，不离开首页  
4）点击“查看详情”进入该菜谱详情页  
5）成功弹出时播放一次轻量提示音  
6）提示音必须：
   - 不是背景音乐
   - 失败态不触发
   - 支持开发者替换素材位
7）首页点击“生成菜谱”成功跳转结果页时，也要播放同类成功提示音  
8）新增音效素材位说明：
   - public/sounds/xxx
   - 配置文件或 constants 中集中管理
9）文档中写明：
   - 音效触发时机
   - 素材替换方式
   - 为什么这属于体验增强而不是核心逻辑
