请重构 RecipeDetailPage，对齐 PRD 2.0。

要求：
1）详情页保留已有信息：
   - 标题
   - 主图
   - 标签
   - 时间
   - 价格
   - 难度
   - 食材
   - 调料
   - 步骤
   - 营养信息
   - 推荐搭配
2）为关键步骤增加辅助图片占位

   - 将当前选中对象加入清单

   - 弹出结果反馈弹窗
7）结果弹窗包含：
   - 已加入数量
   - 对应菜谱名称
   - 查看待购清单
   - 继续浏览菜谱
8）不影响原有收藏逻辑
9）同步更新文档，明确区分：
   - 文件级改动
   - 代码级改动
   - 交互级改动
