# 08_v2_upgrade

本目录用于执行《执行手册2.0》的 **v2 升级归档**：先对当前 1.0 做基线快照（不改业务），再按 v2 的 step 节奏进行增量升级，避免与 1.0 的 step 文档混在一起导致混乱。

## 目录约定

- `00_v1_baseline/`：1.0 基线快照（文件树、路由、localStorage、已知问题、build 记录等）
- `01_prompt_logs_v2/`：v2 每轮 prompt 原文
- `02_step_reports_v2/`：v2 每轮 step report
- `03_issue_reviews_v2/`：v2 issue/反思记录
- `04_flowcharts_v2/`：v2 流程图（`macro_flow_v2.mmd`）
- `05_assets_v2/`：v2 截图与演示素材（before/after）
