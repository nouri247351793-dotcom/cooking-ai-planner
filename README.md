# cooking-ai-planner（大学生做饭学习规划师）

本仓库用于开发课堂作业项目「大学生做饭学习规划师 / 面向大学生的烹饪学习规划工具」。

## 项目原则（来自《执行手册》）

- 技术路线保持：`Vite + React + JavaScript`（`p5.js` 仅做轻量视觉增强，可独立开关/卸载）
- 开发策略：先 `mock` 跑通 MVP，再逐步接入真实 AI
- 评分重点：不仅看成品页面，更看 **vibe coding 过程证据链**（prompt/改动/验证/反思/流程图/截图）

## 过程归档目录

所有过程材料统一放在：

- `2_coding_documentation/`
  - `00_project_brief/`：定位、边界、rubric 映射
  - `01_prompt_logs/`：每轮 prompt 原文
  - `02_step_reports/`：每轮报告（含 why / changed files / key logic / test / issues / next）
  - `03_git_records/`：提交记录索引、分支策略、release notes
  - `04_issue_reviews/`：问题/反思/知识点地图
  - `05_flowcharts/`：宏观流程图（mermaid）
  - `06_assets/`：截图、demo gif、图示
  - `07_exports/`：最终导出的文档占位
